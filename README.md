**Описание**  

Интерактивный тренажёр для обучения операторов работе с клиентами в чате. Приложение имитирует реальный чат-интерфейс с возможностью отвечать на вопросы клиентов и получать обратную связь.

**Основные функции**

- Реалистичный интерфейс чата с аватарами и временем сообщений
- Список активных чатов для переключения между диалогами
- Подсказки с правильными ответами (варианты верных формулировок)
- База знаний (быстрый доступ к документации)
- Адаптивный дизайн для работы на компьютерах и мобильных устройствах
- Панель администратора для пользователей с правами staff

**Установка и запуск**

- Склонируйте репозиторий
- Установите зависимости (если требуется)
- Запустите Django-сервер (py manage.py runserver)  
- Откройте вкладку в браузере
